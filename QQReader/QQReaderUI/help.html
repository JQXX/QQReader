欢迎您使用QQ阅读，支持正版

购买阅点，您需具备以下几点：

  • 您的手机没有越狱
  • QQ阅读是从AppStore下载的
  • 当前苹果账户和下载QQ阅读时的账户为同一账户
  • 苹果账户里有足够的余额用来购买阅点
  • App购买项目的访问限制是打开状态（设置-通用-访问 限制）

苹果账户充值流程：

点击精品推荐最下方自己的Apple ID

选择查看Apple ID

点击充值

选择您想充值的金额并选择银行卡，点击下一步进行充值操作

返回QQ阅读客户端，在我的里选择“我的余额”

选择您想要充值的金额

按照提示，输入您的Apple ID

恭喜！您已完成了苹果帐户的充值

到QQ阅读我的帐户里购买阅点，选择您喜爱的书籍吧
